# LeetCode 周赛

这里存放 LeetCode 周赛和双周赛代码，目录已经统一成按场次编号命名，方便直接定位历史比赛。

## 当前结构

- 每场比赛一个目录，例如 `第491场周赛`、`第177场双周赛`。
- 目录内以题解 `.py` 文件为主，部分场次带有 `复盘.md`。
- [辅助脚本](./辅助脚本/) 用来放赛事相关的小工具，不和正式题解混放。

## 命名约定

- 周赛目录统一使用 `第123场周赛`。
- 双周赛目录统一使用 `第123场双周赛`。
- 复盘统一命名为 `复盘.md`。
- 题解文件保留题号或题名，优先保证可搜索性。

## 说明

- 这里更适合按“比赛”维度回看，不主动再拆回按知识点目录。
- 如果某题里抽出了稳定模板，应迁移到 [../../../模板库](../../../模板库/README.md)。
