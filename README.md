## 算法->游戏

最初接触算法，我的想法很简单：就是在蓝桥杯拿奖，为保研加分。

那时候我只把它当成一个保研加分项，并没有多热爱。最开始在AcWing平台学习，先跟着每日一题打卡，学着学着觉得很有意思，就开始系统攻克基础算法。把基础算法学完后，又觉得内容不够，便开始进阶算法的学习。

进阶算法里有很多内容我难以吃透，只学了一部分就中途停了一段时间，之后就转去刷LeetCode了。

一入LeetCode深似海。

最开始我只是跟着每日一题偶尔写写，刷刷Hot 100、Hot 150。后来遇到的难题越来越多，看题解的时候认识了灵神，发现他整理了很多体系化的题单，像打开了新世界的大门，从此便开始沉浸式刷题，一路坚持了下来。

我的刷题量也慢慢从100题涨到500题、1000题，到现在已经刷完2000题了。

其实刷到后期，对个人能力的提升已经很有限了，就是单纯觉得好玩，就像玩《我的世界》一样——没事就想进去，哪怕什么目标都没有，挖两块煤、搭个小木盒，也觉得很开心。

这个仓库我会持续更新，也会往里放一些有质量的干货内容。不过那些难度过高的算法内容，我大概率也学不会了。
