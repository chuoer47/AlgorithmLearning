# 模板库

这里存放可复用的算法模板和常用代码骨架

## 分类

- 动态规划
- 图
- 字符串
- 搜索
- 数学知识
- 数据结构
- 语法糖

## 适合放进来的内容

- 可以跨题复用的标准模板。
- 常见算法的稳定写法。
- 输入输出、排序、数据结构等高频代码骨架。

## 不适合放进来的内容

- 强依赖某一道题背景的特化解法。
- 还没抽象稳定的临时验证代码。
- 单纯为了存档而保留的某题 AC 代码。

## 维护规则

- 先在 [../刷题记录](../刷题记录/README.md) 完成题目，再把稳定实现回填到这里。
- 同一模板如果保留多个版本，使用语义后缀，例如 `.naive.py`、`.monoqueue.py`、`.v2.py`，不再继续堆叠 `(2)`、`(3)` 这类命名。
